- Pinned からのシーン復元
- セーブデータ影響検知 (UdonSynced フィールド)
- 先方 JSON インポート機能
- CLI ツール (check / update / restore / reassign / import-pinned / show / show-pinned)
- Project Settings UI でパス設定
